    <h1>MacroProgram v1.0.0 배포</h1>
    <p>
      윈도우에서 키보드/마우스 작업을 녹화하고 원하는 횟수로 반복 실행할 수 있는 매크로 프로그램입니다.<br />
      아래 사용 방법을 순서대로 보시면 처음 사용하는 분도 바로 사용할 수 있습니다.
    </p>
  </section>

  <!--소개-->
  <article class="wtl-card">
    <h2>프로그램 소개</h2>
    <p>
      MacroProgram은 반복적인 입력 작업을 자동화하기 위해 만든 윈도우용 도구입니다.
      시나리오를 먼저 만들고, 실제 작업을 한 번 녹화한 뒤, 반복 횟수·반복 간 텀·수행 속도를 설정해 재생할 수 있습니다.</p>
    <p>
      특히 클릭 반복, 동일 문구 반복 입력, 특정 순서 작업 반복 같은 실사용 작업에서 효율이 좋습니다.
    </p>
  </article>

  <!--사용 흐름-->
  <article class="wtl-card">
    <h2>사용 방법 (처음 사용자 기준)</h2>
    <ol class="wtl-step">
      <li>프로그램 실행 후 안내 화면에서 블로그 버튼을 누릅니다.</li>
      <li>왼쪽에서 시나리오를 추가하고 이름/설명을 입력합니다.</li>
      <li>녹화 시작 후 실제 작업을 한 번 수행하고 녹화 중지합니다.</li>
      <li>실행 옆 옵션에서 반복/텀/속도를 설정하고 실행합니다.</li>
      <li>필요하면 파일 메뉴에서 시나리오를 저장합니다.</li>
    </ol>
  </article>
